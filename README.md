# BGRemoveBot

Telegram Bot that uses remove.bg open API

# Setup & Installation:

1. Clone repo to your machine:
```shell
git clone https://github.com/Smarandii/BGRemoveBot
cd BGRemoveBot
```
2. Get Bot token from @BotFather & add your token to .env file
3. Run setup script:
```shell
. install.bin
```
